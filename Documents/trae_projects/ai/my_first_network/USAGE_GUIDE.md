# 智能学习辅助系统使用指南

## 系统概述

智能学习辅助系统是基于OpenAgents框架开发的教育智能体系统，包含三个核心智能体和两个核心模块，为用户提供个性化的学习辅助服务。

### 核心组件

#### 智能体
1. **学情分析Agent**：分析学习数据，识别知识盲点，生成学情报告
2. **资源匹配Agent**：根据学情匹配个性化学习资源
3. **练习优化Agent**：生成个性化练习题，支持分层难度

#### 模块
1. **知识图谱Mod**：构建和分析知识图谱
2. **学习资源库Mod**：管理和匹配学习资源

## 如何使用

### 1. 启动系统

#### 启动网络服务
```bash
openagents network start ./learning_assistant_network
```

#### 启动智能体
```bash
# 启动学情分析Agent
python agents/learning_analysis_agent.py

# 启动资源匹配Agent
python agents/resource_matching_agent.py

# 启动练习优化Agent  
python agents/exercise_optimization_agent.py
```

### 2. 在OpenAgents Studio中连接

1. 打开浏览器，访问 http://localhost:8050
2. 在手动连接区域填写：
   - **主机**：localhost
   - **端口**：8700
   - **使用SSL连接**：取消勾选
3. 点击"连接"按钮

### 3. 核心功能使用

#### 学情分析
- 上传学习数据
- 系统自动分析知识盲点
- 生成详细学情报告

#### 资源匹配
- 提供学情报告
- 获取个性化资源推荐
- 访问推荐的学习材料

#### 个性化练习
- 基于学情生成练习题
- 完成练习并提交
- 获得实时反馈

## 技术细节

### 网络配置
- **网络地址**：localhost:8700
- **gRPC端口**：8600
- **网络ID**：learning-assistant-network-1

### LLM服务
- **服务提供者**：CloseAI
- **模型**：qwen:0.5b-chat
- **API地址**：https://api.closeai-inc.com/v1

### 配置文件
- **网络配置**：`learning_assistant_network/network.yaml`
- **智能体配置**：`agents/`目录下的Python文件
- **LLM配置**：`llm_config.py`

## 常见问题排查

### 1. 网络连接失败

**症状**：无法连接到localhost:8700

**解决方案**：
- 检查网络服务是否正在运行
- 检查端口8700是否被占用
- 重启网络服务

### 2. 智能体连接失败

**症状**：智能体无法连接到网络

**解决方案**：
- 检查网络服务是否正常
- 检查智能体配置是否正确
- 检查gRPC端口8600是否被占用

### 3. LLM服务异常

**症状**：智能体无法生成响应

**解决方案**：
- 检查网络连接
- 切换到本地Ollama备用配置（修改`llm_config.py`）
- 检查API密钥是否有效

### 4. 测试脚本失败

**症状**：运行测试脚本时出现错误

**解决方案**：
- 检查所有智能体是否已启动
- 检查网络服务是否正常
- 检查Python环境配置

## 扩展和定制

### 添加新的智能体
1. 在`agents/`目录下创建新的Python文件
2. 继承`WorkerAgent`类
3. 实现相关方法
4. 配置智能体
5. 启动智能体

### 添加新的模块
1. 在`mods/`目录下创建新的模块目录
2. 实现模块功能
3. 在`network.yaml`中配置模块
4. 重启网络服务
